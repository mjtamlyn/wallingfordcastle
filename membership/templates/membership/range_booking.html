Range bookings

To book a session, select an upcoming club day and find an available slot. You can then select the archers who will be attending and set a distance.

Where possible, please try to use lanes which are already in use that day, and keep longer distances to the left of the range. If you are the first or last person using that lane in the day you will be responsible for getting your target out and putting it away.

This website feature is still quite rough around the edges, if you have any problems please contact Marc. Upcoming features include extending sessions and non-club session bookings.
